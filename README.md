# yan_keras_docker


### pull the docker from docker hub

```bash
docker pull yanliang12/yan_keras:1.0.2
```

### run the docker

```bash
docker run -it ^
-p 0.0.0.0:9971:9971 ^
yanliang12/yan_keras:1.0.2
```

### run the example program

```
http://localhost:9971/notebooks/bidirectional_lstm_imdb.ipynb
```
